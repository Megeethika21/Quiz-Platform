# Coding Quiz Platform 💻

A responsive coding quiz application built using HTML, CSS, and JavaScript.

Users can answer programming-related quiz questions and receive a final score at the end.

## Features
1. Multiple choice quiz questions
2. Instant answer validation
3. Score tracking system
4. Responsive UI
5. Restart quiz functionality
6. Interactive quiz interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Future Improvements
- Timer functionality
- Leaderboard system
- Difficulty levels
- User authentication
- Backend integration with Python Flask
- Database support
